How Custom Layers Work

Custom API sources plug into the processing pipeline. When a book needs metadata, your API is queried alongside built-in sources.

  1. Your API receives book title & author
  2. Response fields are extracted via JSONPath
  3. Extracted data feeds into the book profile with your chosen confidence weight

URL Template Variables:

{{"{{"}}title{{"}}"}}, {{"{{"}}author{{"}}"}}, {{"{{"}}narrator{{"}}"}}, {{"{{"}}isbn{{"}}"}}, {{"{{"}}path{{"}}"}}

JSONPath Examples:

$.title - top-level field
$.results[0].author - first array item
$.data.book.name - nested field
